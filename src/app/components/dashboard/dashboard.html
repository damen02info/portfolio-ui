DeployStream: Laboratorio de Entornos Efímeros

Para demostrar mis competencias integrando Desarrollo Full-stack e Infraestructura (DevOps), he diseñado este laboratorio interactivo conectado a un servidor real en producción. Al interactuar con el panel, disparas un pipeline real que alterará la estética de este portfolio de forma efímera.

1. Disparo Asíncrono: Al pulsar un color, Angular 17+ bloquea la interfaz preventivamente mediante Signals y Spring Boot 3.x delega la carga pesada al instante de forma asíncrona (@Async) invocando la API de Jenkins.
2. Pipeline & Streaming: Jenkins ejecuta un script de automatización local que actualiza los registros en PostgreSQL. Las trazas del proceso se interceptan y se transmiten en tiempo real a la terminal mediante Server-Sent Events (SSE).
3. Mutación y Rollback: Al finalizar con éxito, el fondo de este portfolio cambiará de color reactivamente sin recargar la página. Tras 5 minutos, una tarea programada (@Scheduled) destruirá el entorno temporal restaurando el estado original.
